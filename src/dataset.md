# Mindora Career Guidance Dataset

## Dataset Description

This dataset contains comprehensive career guidance data for AI-powered personalized career pathway recommendations. The data includes user profiles, aptitude test results, personality assessments, skills, experiences, and career outcomes.

## Dataset Columns

### 1. User Profile Features

| Column Name | Type | Description |
|------------|------|-------------|
| user_id | String | Unique identifier for each user |
| name | String | User's full name |
| age | Integer | User's age (16-45) |
| gender | String | Gender (Male/Female/Other) |
| education_level | String | Current education level (12th/Undergraduate/Graduate/Postgraduate) |
| field_of_study | String | Major or field of study |
| cgpa | Float | Cumulative GPA (0-10 scale) |
| college_tier | Integer | College ranking tier (1-3, 1 being top tier) |

### 2. Aptitude Test Results

| Column Name | Type | Description |
|------------|------|-------------|
| math_score | Integer | Mathematics aptitude score (0-100) |
| logical_reasoning_score | Integer | Logical reasoning score (0-100) |
| verbal_ability_score | Integer | Verbal ability score (0-100) |
| analytical_score | Integer | Analytical thinking score (0-100) |
| spatial_reasoning_score | Integer | Spatial reasoning score (0-100) |

### 3. Personality Assessment

| Column Name | Type | Description |
|------------|------|-------------|
| extraversion | Integer | Extraversion level (1-10) |
| conscientiousness | Integer | Conscientiousness level (1-10) |
| openness | Integer | Openness to experience (1-10) |
| agreeableness | Integer | Agreeableness level (1-10) |
| emotional_stability | Integer | Emotional stability (1-10) |
| leadership_potential | Integer | Leadership potential score (1-10) |
| creativity_score | Integer | Creativity assessment (1-10) |
| team_orientation | Integer | Team working preference (1-10) |

### 4. Skills & Experience

| Column Name | Type | Description |
|------------|------|-------------|
| programming_skills | String | Programming languages known (comma-separated) |
| years_experience | Float | Years of work experience |
| internships_completed | Integer | Number of internships completed |
| projects_count | Integer | Number of projects completed |
| certifications | String | Professional certifications (comma-separated) |
| hackathons_participated | Integer | Number of hackathons participated |

### 5. Interests & Aspirations

| Column Name | Type | Description |
|------------|------|-------------|
| career_aspiration_1 | String | Primary career aspiration |
| career_aspiration_2 | String | Secondary career aspiration |
| industry_preference | String | Preferred industry sector |
| work_life_balance_priority | Integer | Priority level for work-life balance (1-10) |
| salary_expectation | String | Salary expectation range |
| relocation_willingness | String | Willingness to relocate (Yes/No/Maybe) |

### 6. Target/Output Variables

| Column Name | Type | Description |
|------------|------|-------------|
| recommended_career_path | String | AI-recommended primary career path |
| career_fit_score | Float | How well the career fits the profile (0-100) |
| estimated_success_probability | Float | Probability of success in recommended career (0-1) |
| skill_gaps | String | Identified skill gaps (comma-separated) |
| recommended_courses | String | Suggested courses/certifications |
| expected_salary_range | String | Expected salary range in INR (LPA) |
