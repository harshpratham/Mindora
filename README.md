
  # Mindora

  Production-hardened app setup for Mindora (React + Vite + Supabase functions).

  ## Quick Start

  1. Install dependencies:
     - `npm i`
  2. Copy env template:
     - `cp .env.example .env`
  3. Fill required variables in `.env`.
  4. Start dev server:
     - `npm run dev`

  ## Quality Commands

  - `npm run lint`
  - `npm run typecheck`
  - `npm run test`
  - `npm run build`
